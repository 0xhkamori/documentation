## Flashing guide
:::warning
You need to make sure that [Custom bootloader (ChouChou)](./custom-bootloader) is installed and working correctly
:::

1. Boot into fastboot mode
2. Flashing logo partition
    ````shell
    fastboot flash logo logo.bin
    ````
