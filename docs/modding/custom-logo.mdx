## Flashing guide
:::warning
At the moment flashing partitions is only possible via mtkclient if you have already unlocked bootloader
:::

1. Install [mtkclient](https://github.com/bkerler/mtkclient) (preferably on linux)
2. Clone repository [fuckyoumoto](https://github.com/moto-penangf/fuckyoumoto)
3. Flashing the logo partition
````shell
$ mtk w logo logo.bin --loader ./sources/fogorow/DA_SWSEC_2316_p325a_dl_forbidden3.bin
````
